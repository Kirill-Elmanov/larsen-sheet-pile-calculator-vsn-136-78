# Программа по расчету шпунта Ларсена (ВСН 136‑78)

Расчет шпунтового ограждения по ВСН 136‑78 с автоматическим подбором глубины заглубления, построением эпюр активного/пассивного давления и изгибающего момента, а также формированием инженерного отчета для защиты перед экспертом главгосэкспертизы.

---

## Практическое применение

Данный инструмент был разработан в рамках прохождения государственной экспертизы по объекту магистрального газопровода в части защиты решений по укреплению стенок котлована шпунтом Ларсена, по требованию эксперта возникла потребность в программе, включающей в себя следующие функции:

- детальный расчет шпунтового ограждения котлована;
- эпюры активного и пассивного давления;
- эпюру изгибающего момента в шпунте;
- расчёт коэффициента запаса по пассивному сопротивлению;
- определление глубины заглубления шпунта;
- проверку и подбор сечения шпунта по моменту сопротивления и напряжениям.

Для экономии времени, снижения риска ручных ошибок и подготовки аккуратного, хорошо структурированного отчета была реализована данная программа как интерактивное кроссплатформенное веб‑приложение.

---

## Функционал

**Основные возможности:**

- Расчет шпунтового ограждения по ВСН 136‑78.
- Учет схемы работы шпунта:
  - консольная схема;
  - схема с одним рядом распорок/анкерных тяг;
  - схема с двумя рядами и т.п.
- Ввод исходных данных:
  - геометрия котлована (глубина, ширина, длина);
  - характеристики грунта (γ, φ, c);
  - положение уровня грунтовых вод;
  - поверхностные нагрузки (распределённые, нагрузка от крана и др.).
- Расчет:
  - коэффициентов активного и пассивного давления;
  - активного давления грунта;
  - пассивного сопротивления;
  - усилий (расчетная глубина заглубления, коэффициент запаса по пассивному сопротивлению);
  - изгибающего момента и поперечной силы в сечении шпунта;
  - требуемого момента сопротивления сечения и напряжений.
- Автоматический подбор глубины заглубления шпунта из условия требуемого коэффициента запаса.
- Подбор профиля шпунта:
  - база типовых профилей (ГОСТ, ArcelorMittal, ThyssenKrupp и др.);
  - сравнение требуемого момента сопротивления с фактическим;
  - оценка запаса и массы выбранного профиля.
- Оценка прогиба шпунтовой стены и сравнение с предельными деформациями.
- Генерация инженерного отчета:
  - исходные данные;
  - промежуточные результаты и формулы;
  - эпюры давления и изгибающего момента;
  - выводы и рекомендации;
  - экспорт в формат для печати, DOCX, Excel, JSON.

## Технологии

- HTML, CSS, JavaScript (single-page application).
- Пошаговая структура интерфейса:  
  1. Ввод исходных данных.  
  2. Расчет коэффициентов.  
  3. Расчет активного/пассивного давления.  
  4. Подбор глубины заглубления.  
  5. Расчет усилий (M, Q).  
  6. Подбор сечения шпунта.  
  7. Проверка деформаций.  
  8. Формирование отчета.
- Визуализация эпюр на основе canvas‑графики.
- Локальное хранение исходных данных и результатов (для построения отчета и повторного использования).

---

## Как пользоваться

1. Открыть приложение в браузере.
2. Ввести:
   - геометрию котлована;
   - характеристики грунта;
   - положение УГВ;
   - поверхностные нагрузки и нагрузку от техники (при наличии).
3. Выбрать схему работы шпунта (консоль, с распорками/анкерами).
4. Запустить расчет:
   - при необходимости воспользоваться автоматическим подбором глубины заглубления;
   - выполнить подбор сечения шпунта из базы профилей.
5. Просмотреть:
   - эпюры активного и пассивного давления;
   - эпюру изгибающего момента;
   - коэффициенты запаса и напряжения в сечении.
6. Сформировать и выгрузить отчет (печать / DOCX / Excel / JSON) для передачи в экспертизу или включения в пояснительную записку.
